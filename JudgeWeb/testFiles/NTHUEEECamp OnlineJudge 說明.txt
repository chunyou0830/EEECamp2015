【小隊員提醒事項】
1. 不要開無痕模式。
2. 題目編號不要任意改動。
3. 每題送出之後需要等 15～40 秒，期間內不要關閉頁面或重新整理。
4. 不要重複點選送出按鈕。
5. 盡量使用 Chrome 或是 Firefox 開啟網頁。

違反任何一項都有可能會 Crash！

----------

【小隊輔注意事項】
1. 如果有小隊員沒辦法通過，請先協助檢查程式是否正確（多印一個 \n 都會影響結果）。
2. 如果 Judge 網頁跳出「Err_XXXXX」，請把這個訊息記錄下來，然後找教學股處理。
3. 如果出現「Internal Server Error」，請等約一分鐘後再試。如果重新嘗試過後還是一樣的訊息，請通報教學股。

該提醒小隊員的事項請不要忘記！不要忘記！不要忘記！很重要所以要講三次！

----------

【教學股注意事項】
1. 如果有人通報「Err_XXXXX」，請先檢查 IDE 和 Browser 設定是不是都正確，然後重新嘗試。如果重試還是不行，請找楊淳佑。
2. 如果有人通報「Internal Server Error」，請先引導小隊員使用沒有 Judge 功備的備用網站：（？），然後告訴楊淳佑。
